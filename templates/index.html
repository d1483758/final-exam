2. 資料庫規劃 (SQLite)

會員資料表 (users)

  • id (INTEGER, PK) - 會員唯一識別碼
  • username (TEXT, UNIQUE) - 帳號
  • email (TEXT) - 電子信箱
  • password (TEXT) - 加密密碼
  • created_at (TIMESTAMP) - 註冊時間

商品資料表 (products)

  • id (INTEGER, PK) - 商品唯一識別碼
  • title (TEXT) - 商品標題
  • description (TEXT) - 商品詳細描述
  • category (TEXT) - 商品分類
  • price (REAL) - 商品價格
  • image (TEXT) - 商品圖片名稱
  • seller_id (INTEGER, FK) - 賣家 ID (關聯至 users.id)
  • created_at (TIMESTAMP) - 刊登時間

3. 網站頁面與路由規劃

  • 🔗 / - 首頁 (商品首頁/瀏覽入口)
  • 🔗 /login - 登入頁面 (F-01)
  • 🔗 /register - 註冊頁面 (F-01)

  • 🔗 /product/edit/<id> - 商品編輯頁面 (F-02)
  • 🔗 /my-products - 我的商品管理頁面 (F-02 新增/修改/刪除主控台)
