2. SQLite 資料表設計

會員表 (users)

  • id (INTEGER, PK) - 會員唯一識別碼
  • username (TEXT, UNIQUE) - 帳號
  • password (TEXT) - 加密密碼
  • email (TEXT) - 電子信箱
  • role (TEXT) - 角色 (user/admin)

商品表 (products)

  • id (INTEGER, PK) - 商品唯一識別碼
  • name (TEXT) - 商品名稱
  • price (REAL) - 商品售價
  • stock (INTEGER) - 剩餘庫存
  • image_url (TEXT) - 圖片網址

購物車明細表 (cart_items)

  • id (INTEGER, PK) - 購物車明細 ID
  • user_id (INTEGER, FK) - 所屬會員
  • product_id (INTEGER, FK) - 商品 ID
  • quantity (INTEGER) - 購買數量

訂單表 (orders & order_items)

  • orders.id (INTEGER, PK) - 訂單編號
  • orders.user_id (INTEGER, FK) - 下單會員
  • orders.total_price (REAL) - 訂單總金額
  • orders.status (TEXT) - 訂單狀態
  • order_items.product_id (INTEGER) - 購買商品
  • order_items.quantity (INTEGER) - 數量
  • order_items.price_at_purchase (REAL) - 購買價格
