💡 新手使用指南

📝

1. 註冊與登入

點選右上角「註冊」新帳號。除了電子信箱，建議填寫您的 LINE ID 等聯絡資訊，這能讓想買您物品的同學更方便聯絡您！

➕

2. 刊登二手商品

登入帳號後，點選上方導覽選單的「刊登商品」。填寫名稱、分類、設定預估售價、附上物資詳細描述，並上傳一張實拍照片即可發布。

🔍

3. 搜尋與瀏覽物資

在首頁橫幅搜尋框輸入關鍵字（如：Python、腳踏車），或前往「所有商品」頁面利用關鍵字與分類篩選，快速找到需要的二手物件。

✉️

4. 聯絡賣家交易

在感興趣的商品詳情頁中，直接查看賣家的電子郵件與 LINE 聯絡帳號，自行以私訊或面交方式安全、無手續費地完成物品交接。
